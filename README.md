# Popel věrnosti — Prolog

Interaktivní textová hra ze světa Warhammer 40 000 / Horus Heresy.
Hraješ za seržanta Gardy smrti během obléhání světa Sundereth.

## ▶ Hrát

**[Spustit hru →](https://petrvana558.github.io/Popel_vernosti/)**

Hra běží přímo v prohlížeči, není potřeba nic instalovat.

## Lokálně

```bash
git clone https://github.com/petrvana558/Popel_vernosti.git
cd Popel_vernosti
```

Pak stačí otevřít `index.html` v prohlížeči (dvojklik nebo přetáhnout do okna).
